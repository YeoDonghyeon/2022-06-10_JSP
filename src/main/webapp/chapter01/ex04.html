JSP를 사용한 동적인 웹페이지

JSP을 사용하면 전달 받은 GET, POST 등의 파라미터 값을 꺼낼 수 있음

DB와 연동해서 필요한 동적인 데이터들을 가져올 수 있음

JSP의 동작 결과를 웹페이지로 보여줄 수 있음

또 JS(JavaScript)를 활용할 수 있고 Jquery도 활용할 수 있음

따라서 ajax도 활용할 수 있음

이런것들을 활용하면 JSP로 동적인 웹페이지를 만들 수 있음

그러나 JSP이 동작하려면 서버가 반드시 필요함

JSP는 HTML, CSS, JS 코드를 입력할 수 있고 자바 코드도 입력할 수 있음 / HTML과 Servlet의 장점들을 합친 것
